- #### Delete the fifth through seventh lines from a file with sed
    - `sed -i '5,7d' ~/.ssh/config`
- ***Notes***
    - `-i` # In place
    - `d` # Delete
    - Delete the fifth line with sed
        - `sed -i '5d' ~/.bash_profile`
    - Delete the last line with sed
        - `sed -i '$d' ~/.bash_profile`

    - `cat -n ~/.ssh/config`
