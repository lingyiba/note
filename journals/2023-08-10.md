- #### Output the multiplication table in C
    - `vi multiplicationTable.c`
      ```
      #include <stdio.h>
      
      int main() {
          int i=0, j=0;
          for(i=1; i<10; i++) {
              for(j=1; j<i+1; j++) {
                  printf("%d*%d=%d\t", j, i, j*i);
              }
              printf("\n");
          }
          return 0;
      }
      ```
- ***Notes***
    - `\t` # Tab character
    - `\n` # Newline character
