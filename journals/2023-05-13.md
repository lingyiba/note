- #### Use Awesome WM
    - `Super-j` # Move cusor
    - `Super-Shift-c` # Kill a program
    - `Super-Enter` # New a terminal
    - `Super-f` # Full screen
- ***Notes***
    - Start awesome automatically
        - `echo "exec awesome" >> ~/.xinitrc`
        - `echo "startx" >> ~/.bash_profile`
