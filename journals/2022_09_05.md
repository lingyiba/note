- [[Git]] 恢复之前版本
  * `git log` # 查看版本号
  * `git revert -n 版本号` # 恢复
  * `git commit -m '注释'` # 提交
  * `git push` # 推送
  * ***注意事项***
  * `版本号` # 一般前七位数即可，不需要输入完整版本号 (举例：`587a881`5e583773fdf2fc5661257808afc0ada6b)
  * ***参考资料***
  * [Git 恢复之前版本的两种方法 reset, revert (图文详解)](https://blog.csdn.net/yxlshk/article/details/79944535)
