- [[Git]] `reset` 恢复之前的某个版本
  * `git reset --hard 8172b02` # reset `8172b02......` 这个版本
  * ***注意事项***
  * `git log` # 查看版本号 (在 `reset` 之前) (版本号举例：`8172b02`) (一般只需要输入前七位即可)
  * `git push -f` 强制推送 (在 `reset` 之后)
  * `git reflog` 查看所有版本号 (包括 `reset` 之后的舍弃的版本)
