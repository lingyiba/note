- [[linux]] `ls -lh`命令
  * `ls` # list (动词：列出)
  * `-lh` # 参数
  * `l` # list (名词：列表)
  * `h` # human readable (人类可读)

- [[linux]] `chmod u+x sleep.py` 命令
  * `chmod` # change mode (切换模式)
  * `u` # user (当前用户)
  * `+x` # 增加 execute (执行) 权限
  * `sleep.py` # 文件名
  * ***参考资料***
  * ![image.png](../assets/image_1662427435091_0.png)
