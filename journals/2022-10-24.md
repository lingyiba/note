- [[HTML]]CSS Internal
  ```html
  <head>
  <style>
  body {background-color:powderblue;}
  h1 {color:red;}
  p {color:red;}
  </style>
  </head>
  <body>
  <h1>Hello Yaoniplan</h1>
  <p>Yaoniplan</p>
  </body>
  ```
  * ***References***
  * ![image.png](../assets/image_1666567690501_0.png)
- [[HTML]]CSS External
  * index.html
  ```html
  <head>
    <link rel="stylesheet" href="styles.css">
  </head>
  <body>
    <h1>Hello Yaoniplan</h1>
    <p>Yaoniplan</p>
  </body>
  ```
  * styles.css
  ```css
  body {
    background-color:powderblue;
  }
  h1 {
    color:blue;
  }
  p {
    color:red;
  }
  ```
