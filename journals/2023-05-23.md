- #### Set prompt string in Linux Bash
    - `PS1='[\u@\h \W]\$ '`
      ```
      [yaoniplan@tux ~]$ 
      ```
- ***Notes***
    - `\` # Escape
        - `u` # User
        - `h` # Hostname
        - `W` # Working directory
    - Enable colors with the ANSI escape codes
        - `\e[` # Escape
        - `32` # Color code
    - Set prompt string permanently
        - `vi ~/.bashrc`
          ```
          export PS1='\n\W\n> '
          ```
        - `source ~/.bashrc`
- ***References***
    - `grep "PS1" ~/.bashrc`
    - `echo $PS1`
    - https://missing-semester-cn.github.io/2020/command-line/ # $PS1
