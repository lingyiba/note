- [[HTML]]Colors RGB
  ```html
  <h1 style="background-color:rgb(255, 0, 0);">Hello Yaoniplan</h1>
  ```
  * ***Notes***
  * `255``0` # intensity (between 0 and 255) of the color
  * `(255, 0, 0)` # (red, green, blue)
  * `rgb(0, 0, 0)` # black
  * `rgb(255, 255, 255)` # white

- [[HTML]]Colors HEX
  ```html

  ```

  * `HEX` # Hexadecimal
  * `rr` # red
  * `gg` # green
  * `bb` # blue
  * `00``ff` # between 00 and ff (same as decimal 0-255)
