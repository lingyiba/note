- #### Search for specific files with the find command
    - `find . -type f -size 1033c ! -executable`
- ***Notes***
    - `f` # File
    - `c` # Bytes

- #### Use rsync to copy file from one server to another
    - `rsync -av test.md yaoniplan@192.168.10.106:/home/yaoniplan/`
- ***Notes***
    - `-a` # --archive
    - `-v` # --verbose
