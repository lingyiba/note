- # Introduction
  JSON (JavaScript Object Notation) is an open standard file format and date interchange format
- # Data Types Syntax
- ## 1. Strings
  `{"name":"John"}`
- ## 2. Numbers
  `{"age":30}`
- ## 3. Objects
  ```
  {
  "employee":{"name":"John", "age":30, "city":"New York"}
  }
  ```
- ## 4. Arrays
  ```
  {
  "employees":["John", "Anna", "Peter"]
  }
  ```
- ## 5. Booleans
  `{"sale":true}`
- ## 6. Null
  `{"middlename":null}`
